ПОЛИТИКА КОНФИДЕНЦИАЛЬНОСТИ
Дата последнего обновления: 22.11.2025
1. ОБЩАЯ ИНФОРМАЦИЯ
Настоящая Политика конфиденциальности описывает, как приложение "Counter" (далее — "Приложение", "мы", "нас", "наш") обрабатывает информацию пользователей.
2. СБОР И ИСПОЛЬЗОВАНИЕ ДАННЫХ
2.1. Данные, которые мы НЕ собираем
Приложение НЕ собирает и НЕ передает следующие данные:
Персональные данные пользователей
Данные о местоположении
Контактную информацию
Данные устройств
Данные о действиях пользователей
Любую другую информацию, передаваемую через интернет
2.2. Локальное хранение данных
Приложение хранит данные только на вашем устройстве:
Настройки счетчиков (названия, единицы измерения, тарифы)
Записи показаний счетчиков
Настройки приложения (валюта, режим закрытия периода)
Все данные хранятся локально с использованием Android DataStore и не передаются третьим лицам или серверам.
3. ДОСТУП К ИНТЕРНЕТУ
Приложение не требует и не использует доступ к интернету. Все функции работают в офлайн-режиме.
4. ПРАВА ПОЛЬЗОВАТЕЛЯ
Вы имеете полный контроль над своими данными:
Все данные хранятся только на вашем устройстве
Вы можете удалить приложение в любой момент, что приведет к удалению всех данных
Мы не имеем доступа к вашим данным
5. ИЗМЕНЕНИЯ В ПОЛИТИКЕ КОНФИДЕНЦИАЛЬНОСТИ
Мы оставляем за собой право вносить изменения в настоящую Политику конфиденциальности. О существенных изменениях мы уведомим пользователей через обновление приложения.
